top命令可用于查看和管控系统进程。   
    

top命令默认以当前消耗的CPU时间对所展示的进程进行排序。  
    

若要对进程进行管控，必须以root用户身份运行top。  
    

top运行画面顶部给出一些系统整体信息。      
    
    
### 常用指令
| 指令  | 作用                                         |
|-------|----------------------------------------------|
| **h** | 查看帮助文档                                 |
| **M** | 改为以内存占用量排序，**P**回到按CPU耗时排序 |
| **1** | 收起/展开CPU占有量显示                       |
| **R** | 逆序展示结果                                 |
| **u** | 输入username，只显示属于某指定用户的进程     |
| **r** | 调整进程优先级                               |
| **k** | 杀死进程                                     |
